Availability

Novocode Toolkit is distributed as Open Source software (Open Source is a trademark of the Open Source Initiative) under the terms and conditions of the Mozilla Public License 1.1.

Since the project has been moved to SourceForge, everything you need can be reached from the Novocode Toolkit summary page, including the latest file releases, access to the current development version via CVS, news items, and a bug tracker.

Note that Novocode Toolkit is now managed as an Eclipse 2.1 project, so if you're using Eclipse 2.1, you can simply import a downloaded release into your workspace or even add :pserver:anonymous@cvs.sourceforge.net:/cvsroot/novocode-tk (with an empty password) to your list of CVS repository locations and check out the module "novocode-tk".
