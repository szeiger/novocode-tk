Using the Toolkit

In order to use the Novocode Toolkit classes with the JDK, simply include lib/tk.jar in your class path.

Example (sh-compatible shell): 
  $ CLASSPATH=$CLASSPATH:/path/to/novocode-tk/lib/tk.jar
  $ export CLASSPATH
Example (csh-compatible shell): 
  $ setenv CLASSPATH $CLASSPATH:/path/to/novocode-tk/lib/tk.jar

The bin/tkjava script should work on most Unices. It can be used to start the examples without requiring a change to the class path.
