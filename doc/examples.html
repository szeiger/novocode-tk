Examples

The examples directory contains the following applications (denoted by their main class):

  • ByteStringTest

    A benchmark for ByteString and ByteStringInputStream.

  • DirExplorer

    Demonstrates the use of TreeModel and FoldedTreeView.

  • NFileTest

    Shows how to use NFiles to navigate a database.

  • DBExplorer

    A DirExplorer that uses NFiles on a database.

  • DBCopy

    Copies files between a database and the local file system.

  • CommandLineConfigTest

    A simple bean which is configured automatically with the CommandLineConfigurator class. Also shows how to use the Version class.
