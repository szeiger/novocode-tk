Compiling the Toolkit

Starting with release 1.4, Novocode Toolkit is built using a build script for ANT. If you have got an official release that you don't want to modify, you can simply use the pre-built binaries and API documentation. If you have made modifications or pulled a version off the CVS repository, you need to run ANT to rebuild the project. Use "ant -projecthelp" to get a list of available build targets.

Since Novocode Toolkit is now managed as an Eclipse 2.1 project, you can also import it into your Eclipse workspace and let Eclipse take care of compiling the source files. The build script can also be run from within Eclipse.
