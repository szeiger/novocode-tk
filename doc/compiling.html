Compiling the Toolkit

A simple Makefile which should work on most Unix platforms is supplied to compile everything. You need to install JBuildTree to use it.

The following make targets are supported:

  • all - Compile all classes
  • alld - Compile all classes with deprecation warnings
  • clean - Remove all compiled class files
  • tk.jar - Create lib/tk.jar
  • dist - Create zip and tar.gz archives for a standard Novocode Toolkit distribution
  • distclean - Remove lib/tk.jar and everything created by "make dist"
  • apidocs - Use javadoc to create the API documentation and post-process it with the beautifier scripts
  • apidocs-clean - Remove API documentation created by "make apidocs"
